Вкладка «Люди»¶

Вкладка люди содержит кнопки на остальные таблицы требуемые для проекта, а именно:

  • Таблица «шестерки»

  • Таблица «пятерки»

  • Таблица «четверки»

  • Таблица «важные»

  • Таблица «лютые»

Как открыть таблицу¶

Для того что бы открыть нужную таблицу откройте в панеле вкладку «люди», после выберите нужную таблицу и нажмите на нее (будет открыто новое окно). В данном окне будет обтображаться информация по данному разделу. Внизу окна доступны три кнопки, работа которых будет описана далее. Работа кнопок будет приведена на примере таблицы «шестёрки». Для остальных таблиц действие кнопок аналогичное.

Добавление человека¶

  1. Нажмите на кнопку добавить, что бы добавить нового сотрудника (будет открыто новое окно)

  2. В открывшемся окне заполните необходимую информацию для создания сотрудника

  3. После создания, нажмите кнопку «ОК» (окно будет закрыто и инфорамция отобразится в таблице)

Изменение человека¶

  1. Нажмите на того сотрудника информацию которого хотите изменить

  2. После нажмите кнопку «Изменить» (будет открыто новое окно)

  3. В окне измените данные тех параметров, которые вам нужны

  4. Нажмите кнопку «ОК» и ваши изменения будут применены

Удаление человека ( не убийство)¶

  1. Нажмите на того сотрудника информацию о котором хотите удалить

  2. После нажмите кнопку «Удалить» (сотрудник будет удален)
